# 📊 Power BI Sales Performance Analytics Dashboard

## Project Overview

This project showcases an interactive **Sales Performance Analytics Dashboard** built using **Power BI**. The dashboard transforms raw sales data into meaningful business insights through KPI tracking, trend analysis, product performance evaluation, category-wise profitability analysis, and regional sales visualization.

The objective of this project is to demonstrate how business intelligence tools can help organizations monitor performance, identify growth opportunities, and support data-driven decision-making.

---

## Key Features

### KPI Cards

The dashboard includes six key performance indicators (KPIs):

* Total Revenue
* Total Orders
* Total Quantity Sold
* Average Order Value (AOV)
* Total Profit
* Total Margin %

These KPIs provide an executive-level overview of overall business performance.

---

### Interactive Visualizations

#### Monthly Revenue Trend

* Tracks revenue performance over time.
* Helps identify seasonal trends and sales growth patterns.

#### Top 10 Products by Revenue

* Highlights the highest revenue-generating products.
* Enables product performance analysis.

#### Revenue by Category

* Displays revenue contribution by product category.
* Connected to a Category Slicer for dynamic filtering.

#### Profit by Category

* Shows profitability across different categories.
* Updates automatically based on selected category filters.

#### Revenue by Region

* Visualizes sales distribution across regions.
* Connected to a Region Slicer for interactive analysis.

---

## Dashboard Interactivity

### Category Slicer

Allows users to filter and analyze:

* Revenue by Category
* Profit by Category

### Region Slicer

Allows users to:

* Filter regional sales data
* Analyze region-specific revenue performance
